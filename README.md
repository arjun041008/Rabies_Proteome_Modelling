# Interpretable Predictive Modeling of Host Specificity of Rabies lyssavirus from Glycoprotein Sequences

Predicting Rabies Virus Hosts from Protein Sequences

This project explores how machine learning can be used to predict the host species of Rabies lyssavirus (RABV) directly from its glycoprotein (G protein) sequence. The focus is not only on building accurate predictive models, but also on understanding which parts of the protein sequence influence host specificity.

Based on the paper: Interpretable Predictive Modeling of Host Specificity of Rabies lyssavirus from Glycoprotein Sequences

Overview

Rabies is a zoonotic virus with an extremely high fatality rate once symptoms appear. It infects a wide range of mammals, including humans, dogs, bats, and foxes. Because the virus evolves rapidly, identifying the host associated with a given viral strain is important for tracking transmission and improving vaccine strategies.

This project approaches the problem by learning a mapping from amino acid sequences of the viral glycoprotein to host species labels. The glycoprotein is particularly informative because it is exposed on the viral surface and plays a central role in host interaction.

Approach

Two main modeling pipelines were developed.

The first uses protein embeddings generated by ProtBERT, followed by dimensionality reduction with PCA. These features are then used to train classical machine learning models such as logistic regression, random forests, and support vector machines.

The second pipeline encodes amino acid sequences directly as numerical inputs and uses deep learning models, including convolutional neural networks (CNNs) and recurrent neural networks (GRU and LSTM), to capture sequential patterns in the data.

Among all models tested, the LSTM achieved the highest performance, with an accuracy of approximately 90 percent. This highlights the importance of sequence order in determining host specificity.

Interpretability

A central goal of this project is to move beyond black-box predictions and understand how the models make decisions.

Several interpretability techniques were applied:

Saliency maps were used to identify which residue positions most strongly influence predictions
In-silico mutagenesis was performed to measure how changes at specific residues affect model outputs
Hidden state representations from the LSTM were visualized using UMAP to examine how sequences cluster by host

These methods provide insight into both global and residue-level patterns learned by the models.

Key Findings

The models successfully recover known biologically important regions, including established antigenic sites of the glycoprotein.

In addition, a strong and consistent signal was identified in the residue range 460–520. This region lies outside the traditionally emphasized antigenic domains and may play a role in host-specific viral behavior, possibly related to cellular entry or interaction with internal viral components.

This result suggests that host specificity is influenced by a broader set of structural and functional features than previously assumed.

Tools and Technologies

The implementation uses:

Python
TensorFlow and Keras for deep learning
Scikit-learn for classical machine learning
ProtBERT for protein embeddings
UMAP for visualization of learned representations
Significance

This project demonstrates how machine learning can be applied to biological sequence data in a way that is both predictive and interpretable. By identifying residue-level patterns associated with host specificity, it contributes to a deeper understanding of virus–host interactions.

The approach can be extended to other viruses and may support applications in vaccine design, surveillance, and computational virology.
